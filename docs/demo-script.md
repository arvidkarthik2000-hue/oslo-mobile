# OSLO Demo Script — May 2, 2026

## Prerequisites
- Signed APK installed on demo Android device
- Backend running (uvicorn on Supabase Mumbai)
- AI endpoint live (verify: `curl https://oslo-ai.<tunnel>.trycloudflare.com/health`)
- Device connected to WiFi

## Demo Flow (10-12 minutes)

### 1. First Launch (30s)
- Open OSLO → splash screen "Setting up your demo..."
- App loads to Home tab with pre-populated demo data
- Point out: greeting, recent activity cards, values to watch
- **Talking point:** "All of this was extracted from uploaded lab reports using AI."

### 2. Upload a Real Lab Report (60s)
- Tap **📄 File** quick action
- Pick a real Indian lab report PDF (from phone files)
- Show the processing: "Classifying and extracting..."
- Review the extraction table — point out LOINC codes, ref ranges, status pills
- Tap **Confirm & Save**
- **Talking point:** "Under 10 seconds, fully structured. No manual data entry."

### 3. Check Home (15s)
- Back on Home — show the new card in Recent Activity
- Show Values to Watch if any flags were detected

### 4. Timeline (30s)
- Switch to **Timeline** tab
- Show date-threaded events
- Use filter chips: Lab → Rx → All
- Tap the **+** button → write a quick text note → save
- **Talking point:** "Every health event, one timeline. Add your own notes too."

### 5. Records (15s)
- Switch to **Records** tab
- Show category counts: Lab Reports (X), Prescriptions (X)
- Tap a category → see the filtered list

### 6. Lab Report Detail + Explain (45s)
- Open a lab report from Records
- Show the value table with ref ranges and status dots
- Tap **✨ Explain this report**
- Wait for AI explanation → show the patient-friendly narrative
- **Talking point:** "LabSamjho-style explanation, generated by AI, in plain English."

### 7. Trends (30s)
- Switch to **Trends** tab
- Show MetricCard grid with sparklines
- Point out: "HbA1c trending up over 3 reports, cholesterol improving"
- Use filter chips: Cardiovascular → Metabolic
- **Talking point:** "See how your health changes over time, at a glance."

### 8. Smart Report (45s)
- Go to Home → tap **🧠 View your Smart Report**
- Show the loading state → then system cards (Cardiovascular, Liver, Renal...)
- Expand a system card to show key values
- Tap **📤 Share with doctor**
- **Talking point:** "One AI-generated summary across all your records, shareable as PDF."

### 9. Ask AI (45s)
- Switch to **Ask** tab
- Type: "What was my last HbA1c?"
- Show the AI response with citation
- Type: "Is my cholesterol dangerous?"
- Show the refusal + redirect to doctor
- **Talking point:** "Smart enough to answer, responsible enough to decline."

### 10. Emergency Profile (15s)
- Go to Home → tap **🆘 Emergency Profile**
- Show: name, blood group, allergies, conditions, medications, emergency contact
- **Talking point:** "In an emergency, all critical info in one screen."

### 11. Subscribe Flow (15s)
- Tap the subscribe banner on Home
- Show the Family Plan benefits and pricing
- Tap Subscribe → show test mode success
- **Talking point:** "Freemium model, ₹199/month for families."

## Recovery Steps

### If AI endpoint is down:
- App shows "Processing queued" → explain this is the degraded mode
- Show cached Smart Report if available
- Switch to demo sections that don't need AI (Timeline, Records, Emergency)

### If upload fails:
- Show "Upload failed, tap to retry" error state
- Try with a different file
- If persistent, switch to pre-loaded demo data

### If app needs a fresh start:
- Settings → **Reset to Demo State** → restores all pre-loaded data
- Settings → **Reset to Empty State** → clean slate for live upload demo

### If nothing works:
- Show the pre-recorded screen recording from Day 8
- Located at: `assets/demo-recording.mp4`
